# Calculate Single Probability Component

Calculate Single Probability Component

## Usage

``` r
prob2.single(K, W2i, Xi.long, tau, Di.short, Xi.short, Zi, t0, covariate.value)
```

## Arguments

- K:

  Kernel weights.

- W2i:

  IPCW weights.

- Xi.long:

  Long term event times.

- tau:

  Prediction window.

- Di.short:

  Short term event indicators.

- Xi.short:

  Short term event times.

- Zi:

  Covariates.

- t0:

  Landmark time.

- covariate.value:

  Covariate value.
