# Calculate AUC for Landmark Prediction

Calculate AUC for Landmark Prediction

## Usage

``` r
AUC.landmark(t0, tau, data, short = TRUE, weight = NULL)
```

## Arguments

- t0:

  Landmark time.

- tau:

  Prediction window.

- data:

  Data frame.

- short:

  Logical.

- weight:

  Optional weights.
