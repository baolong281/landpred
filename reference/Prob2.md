# Calculate Probability Component (No Kernel)

Calculate Probability Component (No Kernel)

## Usage

``` r
Prob2(t0, tau, data, covariate.value, weight = NULL)
```

## Arguments

- t0:

  Landmark time.

- tau:

  Prediction window.

- data:

  Data frame.

- covariate.value:

  Covariate value.

- weight:

  Optional weights.
